流程操作，扩展的业务动作

1.案卷挂起


2.补正告知，自案卷受理日期起5个工作日内才能发起补正告知。

1)发起补正时可选择案卷当前的材料清单， 作为补正材料列表，或手动添加。默认状态未“待确认”
2)补正受理时，需按发起时的材料清单上传附件并确认。未确认情况下，不允许补正受理。
3)打印补正告知回执单(“待确认”的材料)
4)补正受理时允许再次发起补正,添加新的补正材料清单。


3.特别程序
1)特别程序事项：包括事项名称、事项种类、事项时限
2)批准人： 先采用人员选择，可选范围为当前操作者所属部门下的所有人员。
后期优化审批事项表存在一个明细表，记录各个审批事项的特别程序批准人，审批事项编号存储在业务定义审批事项字段(ConceptName.Relation)中。
